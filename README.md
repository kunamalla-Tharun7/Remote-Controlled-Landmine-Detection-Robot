# 🔍 Remote Controlled Landmine Detection Robot

## 📌 Project Overview
The **Remote Controlled Landmine Detection Robot** is designed to detect hidden landmines in hazardous areas without risking human life. This robot is operated remotely and uses sensors to identify metallic objects buried underground.

---

## 🎯 Objective
- Ensure **human safety** in mine-affected areas  
- Provide a **low-cost and efficient solution** for landmine detection  
- Assist military and rescue operations  

---

## ⚙️ Key Features
- 🤖 Remote controlled operation  
- 🔎 Metal detection sensor  
- 🚨 Alert system (Buzzer/LED)  
- 🔋 Battery powered  
- 🚗 Mobility for rough terrains  

---

## 🛠️ Technologies Used
- Microcontroller (Arduino)  
- Metal Detector Sensor  
- Wireless Communication (RF / Bluetooth)  
- Motor Driver (L293D / L298N)  
- DC Motors  
- Power Supply  

---

## 🧠 Working Principle
1. The robot is controlled remotely by the user  
2. It moves using motors across the terrain  
3. The metal detection sensor scans the ground continuously  
4. When metal is detected:  
   - Buzzer/LED is activated  
   - User gets alert
